# 🌌 Attractor Explorer v20250309

An interactive 3D explorer for chaotic attractors and dynamical systems. Rotate, tweak parameters, and watch complex patterns emerge in real time.

---

## ✨ Features

- 🖱️ **Screensaver Mode**  
  Double-click anywhere to toggle a minimal, idle-friendly display.

- 🌀 **3D Interaction**  
  Click + drag (or swipe) to rotate the attractor in 3D space.

- ⚙️ **Multiple Systems**  
  Explore **Lorenz**, **Halvorsen**, and other chaotic systems.  
  - Each system includes interactive presets and tunable parameters.

- 📈 **Real-Time Simulation**  
  Uses the **Runge-Kutta method** to solve differential equations on the fly.

- 🎨 **Retro Mode**  
  Pixelated, low-res visuals with a color scheme inspired by  
  *X-COM: Terror from the Deep*.
